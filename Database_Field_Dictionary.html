用户系统

Member 1
users — 用户账号
字段名类型必填说明
idUUID自动主键，由数据库自动生成，全局唯一
emailString必填登录邮箱，全局唯一，不可重复注册
passwordString必填bcrypt 加密后的密码，原始密码永远不存储
usernameString必填唯一用户名，用于 @mention 和个人主页 URL，不可重复
display_nameString?可空公开展示名称，可以重复，允许中文
avatar_urlString?可空头像图片的 Cloudinary 云端 URL
bioString?可空个人简介，建议最长 300 字
roleRole默认 user用户角色，控制页面访问权限，见枚举说明
created_atDateTime自动账号注册时间
updated_atDateTime自动账号最后更新时间，由 Prisma 自动维护

关注关系

Member 3
user_follows — 用户之间的关注记录
字段名类型必填说明
idUUID自动主键
follower_idUUID必填发起关注的用户 ID（关注别人的人）
following_idUUID必填被关注的用户 ID（被别人关注的人）
created_atDateTime自动关注时间

救助机构 / 诊所

Member 5
organizations — 救助机构和诊所账号
字段名类型必填说明
idUUID自动主键
nameString必填机构名称，例如「爱猫救助站」
typeOrgType必填机构类型：rescue（救助机构）或 clinic（诊所）
emailString必填机构登录邮箱，全局唯一
passwordString必填bcrypt 加密后的机构账号密码
phoneString?可空联系电话
addressString?可空机构地址，用于地图展示或线下联系
logo_urlString?可空机构 Logo 的 Cloudinary 云端 URL
descriptionString?可空机构简介
license_numberString?可空执照编号或注册编号，用于平台资质核实
is_verifiedBoolean默认 false是否已通过平台认证，认证后显示认证标记
created_atDateTime自动注册时间
updated_atDateTime自动最后更新时间

领养活动

Member 5
adoption_events — 领养会/领养活动记录（对应 Excel 的「届」）
字段名类型必填说明
idUUID自动主键
nameString必填活动名称，例如「第 2 届领养会」
editionInt?可空届次编号，例如 2、3、4，用于排序和筛选
start_dateDateTime必填活动开始日期
end_dateDateTime?可空活动结束日期，单日活动可不填
locationString?可空活动地点
descriptionString?可空活动说明和注意事项
org_idUUID?可空主办机构 ID，关联 organizations 表
created_atDateTime自动创建时间
updated_atDateTime自动最后更新时间

猫咪档案

Member 2
cats — 猫咪主档案
字段名类型必填说明
idUUID自动主键
nameString必填猫咪名字
breedString?可空品种，例如「奶橘」「狸花」「安哥拉」
age_monthsInt?可空年龄（以月为单位），例如 6 = 6个月，24 = 2岁
genderCatGender?可空性别枚举：male（公）/ female（母）/ unknown（未知）
colorString?可空毛色描述，例如「橘白」「黑白」
descriptionString?可空猫咪简介，性格、习惯、特点等
photo_urlString?可空主照片的 Cloudinary 云端 URL
statusCatStatus默认 available当前状态：available（可领养）/ adopted（已领养）/ fostered（代养中）/ deceased（已离世）
is_neuteredBoolean?可空是否已完成绝育/结扎。null 表示不明
is_vaccinatedBoolean?可空是否已接种核心疫苗。null 表示不明
is_dewormedBoolean?可空是否已完成驱虫（体内+体外）。null 表示不明
intake_dateDateTime?可空猫咪进入系统的日期，即救助日期或登记日期
found_locationString?可空发现/救助地点，流浪猫被发现的地址描述，例如「台北市大安区和平东路」
owner_idUUID?可空猫主人的用户 ID，个人养猫用户关联
org_idUUID?可空来源机构的 ID，流浪猫由救助机构登记时关联
event_idUUID?可空当前参与的领养活动 ID
created_atDateTime自动档案创建时间
updated_atDateTime自动档案最后更新时间
cat_tags — 猫咪性格/特点标签
字段名类型必填说明
idUUID自动主键
cat_idUUID必填关联猫咪 ID
tagString必填标签内容，例如「亲人」「怕生」「爱玩」「安静」「粘人」
created_atDateTime自动添加时间
cat_requirements — 猫咪领养要求
字段名类型必填说明
idUUID自动主键
cat_idUUID必填关联猫咪 ID
descriptionString必填要求描述，例如「需要有独立房间」「不可与其他宠物同住」
created_atDateTime自动创建时间
cat_updates — 猫咪动态更新
字段名类型必填说明
idUUID自动主键
cat_idUUID必填关联猫咪 ID
contentString必填动态内容，例如「今天打了疫苗」「体重增加到 3kg」「已成功领养」
photo_urlString?可空配图的 Cloudinary 云端 URL
created_atDateTime自动发布时间

领养模块

Member 2
adopter_preferences — 用户领养偏好设置
字段名类型必填说明
idUUID自动主键
user_idUUID必填关联用户 ID，每位用户只有一份偏好设置
preferred_ageString?可空偏好年龄段，例如「kitten」「adult」「senior」
preferred_genderString?可空偏好性别：male / female / no_preference
preferred_breedString?可空偏好品种，允许为空表示不限
accept_special_needBoolean?可空是否愿意领养特殊需求猫咪（残疾、慢性病等）
home_typeString?可空居住类型，例如「apartment」「house」，用于匹配猫咪要求
has_other_petsBoolean?可空是否已有其他宠物，用于匹配猫咪要求
has_childrenBoolean?可空是否家中有小孩，用于匹配猫咪要求
created_atDateTime自动创建时间
updated_atDateTime自动最后更新时间
adoption_applications — 领养申请
字段名类型必填说明
idUUID自动主键
user_idUUID必填提交申请的用户 ID
cat_idUUID必填申请领养的猫咪 ID
event_idUUID?可空通过哪次领养活动提交的，线上直接申请可为空
statusApplicationStatus默认 pending申请状态：pending（待审核）/ approved（通过）/ rejected（拒绝）
messageString?可空申请人的自我介绍或申请说明，给机构参考
reviewed_byUUID?可空审核人的用户 ID（rescue_staff），审核前为空
reviewed_atDateTime?可空审核完成的时间，审核前为空
reject_noteString?可空拒绝原因说明，仅在 status=rejected 时填写
created_atDateTime自动提交申请时间
updated_atDateTime自动最后更新时间

社区模块

posts — 社区帖子

idUUID自动主键

contentString必填帖子正文，不限长度
image_urlString?可空帖子配图的 Cloudinary 云端 URL

字段名类型必填说明
idUUID自动主键
user_idUUID必填评论用户 ID
post_idUUID必填被评论的帖子 ID
contentString必填评论文字内容
created_atDateTime自动评论时间

健康管理模块

Member 5
owner_health_records — 猫主人记录的健康档案
字段名类型必填说明
idUUID自动主键
cat_idUUID必填关联猫咪 ID
user_idUUID必填记录人（猫主人）的用户 ID
record_typeHealthRecordType必填记录类型枚举：vaccine / deworming / checkup / treatment / surgery / other
descriptionString必填详细描述，例如「接种狂犬疫苗，第一针」
dateDateTime必填该记录发生的实际日期（就诊日期或记录日期）
next_due_dateDateTime?可空下次需要处理的日期，例如疫苗补打日期，用于提醒功能
weight_kgFloat?可空记录时猫咪体重（千克），例如 4.2
vet_nameString?可空就诊医生姓名
clinic_nameString?可空就诊诊所名称
created_atDateTime自动记录创建时间
updated_atDateTime自动最后更新时间
clinic_health_reports — 诊所上传的官方健康报告
字段名类型必填说明
idUUID自动主键
cat_idUUID必填关联猫咪 ID
org_idUUID必填上传报告的诊所机构 ID
report_typeClinicReportType必填报告类型枚举：vaccination / deworming / checkup / blood_test / treatment / surgery / other
descriptionString必填报告详细描述，必填
file_urlString?可空报告附件 URL（PDF 或图片扫描件）
dateDateTime必填报告对应的实际日期（检查日期，非上传日期）
created_atDateTime自动上传时间
updated_atDateTime自动最后更新时间（允许诊所补充修改）
health_share_permissions — 健康数据共享授权
字段名类型必填说明
idUUID自动主键
cat_idUUID必填被授权查看数据的猫咪 ID
user_idUUID必填做出授权决定的猫主人用户 ID
org_idUUID必填被授权查看数据的诊所机构 ID
is_allowedBoolean默认 false是否允许该诊所查看该猫的健康数据，false=拒绝，true=允许
created_atDateTime自动首次设置时间
updated_atDateTime自动最后修改授权时间

聊天功能

Member 6
conversations — 聊天会话（用户与机构之间）
字段名类型必填说明
idUUID自动主键
user_idUUID必填发起对话的普通用户 ID
org_idUUID必填对话对象的机构账号 ID（救助机构或诊所）
created_atDateTime自动对话创建时间，同一用户与同一机构只有一条对话记录
messages — 聊天消息
字段名类型必填说明
idUUID自动主键
conversation_idUUID必填所属对话 ID
sender_idUUID必填发送者的用户 ID（可以是普通用户或机构账号）
contentString必填消息文字内容
created_atDateTime自动发送时间
message_attachments — 消息附件
字段名类型必填说明
idUUID自动主键
message_idUUID必填关联消息 ID
file_urlString必填附件的 Cloudinary 云端 URL
file_typeString必填附件 MIME 类型，例如「image/jpeg」「application/pdf」
created_atDateTime自动上传时间

枚举值完整说明

Role — 用户角色
user普通用户/猫主人，可发帖、申请领养、管理健康记录
rescue_staff救助机构工作人员，可录入猫咪、审核领养申请、查看数据分析
clinic_staff诊所工作人员，可上传官方健康报告，查看授权猫咪记录
admin系统管理员，拥有全部权限（当前阶段留空备用）
OrgType — 机构类型
rescue救助/领养机构，收容流浪猫、主办领养活动、审核领养申请
clinic动物诊所，上传官方健康报告、查看授权的猫咪健康数据
CatGender — 猫咪性别
male公猫
female母猫
unknown性别未知（幼猫或尚未确认）
CatStatus — 猫咪当前状态
available可领养，在领养列表中展示
adopted已领养，不再展示但保留档案
fostered代养中，暂时不对外开放领养
deceased已离世，仅作历史记录保留
ApplicationStatus — 领养申请状态
pending待审核，等待机构工作人员处理
approved已通过，猫咪状态应同步更新为 adopted
rejected已拒绝，应填写 reject_note 说明原因
HealthRecordType — 健康记录类型（主人记录）
vaccine疫苗接种，填 next_due_date 提醒补打
deworming驱虫处理（体内/体外），填 next_due_date 提醒下次
checkup定期体检，无特殊后续
treatment疾病治疗记录，例如感冒、骨折
surgery手术记录，例如绝育手术
other其他健康相关记录
ClinicReportType — 诊所报告类型
vaccination官方疫苗接种证明
deworming官方驱虫证明
checkup常规体检报告
blood_test血液检查报告
treatment疾病治疗记录
surgery手术记录（含绝育手术证明）
other其他官方医疗文件
